# 微信城市群聊与红娘牵线服务网站

## 功能介绍

本网站提供全国各城市的微信付费群聊服务和红娘牵线服务，用户可以选择自己所在的省市县，然后选择需要的服务类型进行支付。支付完成后，用户可以添加客服微信获取相应服务。

## 网站特点

- **全国城市覆盖**：支持全国主要省市县的选择
- **两种服务类型**：城市群聊服务和红娘牵线服务
- **统一价格**：所有服务统一收费9.99元
- **微信支付**：对接第三方支付接口，支持微信支付
- **响应式设计**：兼容手机、平板和电脑等各种设备

## 文件结构

- `index.html`：网站主页面
- `styles.css`：样式文件，实现响应式设计
- `cities.js`：城市数据和城市选择联动功能
- `script.js`：支付处理逻辑和用户交互
